# BTS PROJECT

## Intro to git
- git is a version control system

### initialize a git repository
```bash
git init 
```

### Add files to staging area
```bash
git add <filename>
```

### list branchs
```bash
git branch
```
### create a new branch
```bash
git checkout -b <branchname>
```
### return to another branch
```bash
git checkout <branchname>
```
